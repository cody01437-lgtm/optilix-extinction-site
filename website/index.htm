Server Administration

Admin Commands

/function extinction:admin/grant_admin

Grants yourself admin access. Only useful if you're already an op since regular players can't run /function commands anyway.

/function extinction:admin/revoke_admin

Removes your own admin access.

/function extinction:admin/start Core

Resets everything to Phase 1 Day 1 and begins the event. Bossbars appear, effects start.

/function extinction:admin/stop Core

Halts the event completely. Clears all player effects, kills all withers and phantoms, resets time to day, hides bossbars, and zeroes all scoreboards. Use this if something goes wrong or you want to reset.

/function extinction:admin/skip_phase Testing

Immediately jumps to the next phase. Useful for testing or if you want to manually pace the event faster than one phase per day.

/function extinction:admin/set_phase_1 through /function extinction:admin/set_phase_10 Testing

Jumps directly to any specific phase and triggers its start effects. Useful for testing individual phases or jumping ahead dramatically mid-event.

/function extinction:admin/status Info

Shows a private status panel only you can see with the current phase name, active/inactive status, and ticks elapsed in the current day. Also lists all available commands as a reminder.

/function extinction:admin/info Info

Shows an information text in chat. Also displays credits.
